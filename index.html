Dados coletados pelo App

O aplicativo Engenharia Social - Conscientize-se coleta os seguintes dados: data de nascimento, escolaridade e gênero. Além destes são coletados dados referentes a realização de testes de conhecimento realizados pelo usuário. Tais dados são atrelados de forma anônima a uma conta de usuário criada automaticamente pelo aplicativo e cadastrada no banco de dados. Em outra palavras os dados são armazenados de maneira que não é possível identificar o autor dos mesmos.

Uso dos dados

Os dados coletados serão utilizados para fins científicos com o intuito de aprimorar o conteúdo do app e análisar o desempenho dos usuários.
